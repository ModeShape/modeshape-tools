ModeShape Tools - CND Editor

Child Nodes Table

The CND Editor's Child Nodes Table displays the child node definitions for the selected node type definition. A child node definition can be edited using the Child Node Editor.

Here is what the Child Nodes Table looks like:

The table's toolbar contains buttons that do the following:

Add Child Node Definition  opens the Child Node Editor which allows you to create a new child node definition which is then added to the selected node type definition
Edit Child Node Definition  opens the Child Node Editor which allows you to edit the selected child node definition
Delete Child Node Definition  deletes the selected child node definition from the selected node type
Copy Child Node Definition  copies the selected child node definition to the system clipboard
Paste Child Node Definition  pastes into the selected node type definition a child node definition from the system clipboard
Show Inherited Child Node Definitions  indicates if child node definitions inherited by the selected node type definition are displayed (only from node types within this CND or from built-in CNDs are currently supported)
Open Declaring Node Type Definition  selects the node type definition that declares the selected child node definition (if declared within the same CND)

Copied child node definitions can be pasted into the same CND editor or into a different CND editor.
