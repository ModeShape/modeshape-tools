Namespace Editor

The Namespace Editor is used to create or modify a CND's namespace mapping.

Here is what the Namespace Editor looks like:

The dialog consists of the following controls:

Header Section  shows information and validation error messages
Prefix  a unique prefix that identifies this namespace mapping
URI  a unique URI that identifies this namespace mapping
Comments  additional information about this namespace mapping that is saved to the file

Once all required information is filled in and validated, the 'Finish' button will be enabled and the namespace mapping can be created or changed. The new or changed namespace mapping will appear in the CND editor's namespaces table. The Namespace Editor can be opened from the CND editor's namespaces table by either:

  • clicking the Add Namespace toolbar button, or
  • selecting a namespace mapping and clicking the Edit Namespace toolbar button
